  <h1>🛡️ Woman Security System</h1>
  <p><span class="badge">AI-Powered | Real-Time Risk Detection | Privacy-First</span></p>

  <div class="section">
    <h2>🔍 Overview</h2>
    <p>Woman Security System is a smart chat platform that detects high-risk, scam-like, or threatening interactions in real-time using NLP and AI. Designed for women's digital safety, it provides early warnings and logs incidents securely.</p>
  </div>

  <div class="section">
    <h2>⚙️ Technologies Used</h2>
    <ul>
      <li>Python 3.10+</li>
      <li>Flask / FastAPI for backend</li>
      <li>SpaCy / Transformers for NLP</li>
      <li>React / Streamlit for chat interface</li>
      <li>MongoDB / Firebase for data storage</li>
    </ul>
  </div>

  <div class="section">
    <h2>📦 Installation</h2>
    <pre><code>git clone https://github.com/Asarafhack/Woman-Security-System.git
cd Woman-Security-System
python -m venv venv
source venv/bin/activate  # On Windows: venv\\Scripts\\activate
pip install -r requirements.txt</code></pre>
  </div>

  <div class="section">
    <h2>🚀 How to Run</h2>
    <pre><code># CLI version
python app.py

# Streamlit version (if implemented)
streamlit run app.py</code></pre>
  </div>

  <div class="section">
    <h2>📤 Output</h2>
    <ul>
      <li>⚠️ Flags suspicious messages (keyword + NLP-based context)</li>
      <li>📋 Shows alerts in the chat UI</li>
      <li>📁 Stores logs with timestamps</li>
      <li>📲 Optional alert triggers to emergency contacts</li>
    </ul>
